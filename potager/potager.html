This game is made for 1 player.
Default board is 3x3 squares, with at most 4 cyan bubles on each square.
The state where all bubles are present is called the "initial state".
You can activate a square by clicking on it, or using numpad/digit keys.

Goal:

Automatic: random moves are made, then try to go back to the initial board (every buble is active, cyan). Each time you get it right, the game becomes harder (every parameter can be changed: random, colors, size, bubles).
Figure: go from initial state to a particular state (imposed).
Figure, revert: go back to initial state, from a particular state.

When you click on a square, let's call it A, it finds all it's neighbors, let's call one of them B.

The board is like a ring: for example, if A is at the top, "above A" will mean "at the bottom, on the same column".

If there is an arrow on A pointing to B, let's call it C, then the arrows in B will change color except the arrow on B pointing to A.

More precisely, if there is more than 2 colors, the value of C will be added to the changed arrows on B.

For more than 2 colors, 0:black 1:cyan 2:yellow 3:magenta 4:lime 5:blue 6:red 8:white

If you press the "restart" button, you will be on the same board with the same random moves
